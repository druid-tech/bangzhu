如何将数据同步到 Movebank？

您可以将设备采集的数据同步到 Movebank 以进行进一步分析、研究和共享。开始之前，请确保您已经在 Movebank 创建了一个 Study，且为该 Study 创建了至少一个生物。关于创建 Study 和生物的具体步骤，请参见 Movebank 在线帮助手册： https://www.movebank.org/cms/movebank-content/manual。

  1. 在 DEBUT 数据中心菜单栏单击 “个人中心” > “我的信息”。

  2. 在 Movebank 区域点击 图标设置您的 Movebank 账号。
    输入您的 Movebank 用户名并单击 “确认”。
    注意： DEBUT 数据中心每次只能同步数据到一个 Movebank 账号。

  3. 登录 Movebank 并导航至您的 Study。

  4. 在 Live Feeds 下拉列表中选择 Druid GSM。

    Movebank 进入 live feeds 管理页面。

  5. 单击 New 添加新的 Tags.
    在 Add/Remove Tags 窗口，选择可用的 tag 然后单击 Add all Tags.

    完成后单击 Save。
    您可以将 Druid GSM 添加到多个 Study。

  6. 返回 Study 页面。

  7. 在 Manage 下拉列表中选择 Deployment Manager。
    如果您没有看到 Deployment Manager 选项，请在 24 小时后再次尝试。

    Movebank 进入 Manage Deployments 页面，该页面与下图的例子相似：

    该页面所示的 Tag ID 对应 DEBUT 数据中心的 设备编号.
    该页面所示的 Time of data point 对应 DEBUT 数据中心的 采集时间。

  8. 将鼠标悬停在一个 Tag 上然后单击 Add animal.

    Add Animal to Tag 窗口弹出。

  9. 在下拉列表中选择一个生物。

  10. 填写相关信息然后单击 Save。

    您已经成功将该 Tag 部署到了生物。

注意

  • DEBUT 数据中心会在每天的 UTC 时间 00:00 点同步数据。
  • 当您在 DEBUT 数据中心首次设置 Movebank 账号时，数据中心会将所有数据同步到 Movebank。
  • 当您在 DEBUT 数据中心修改 Movebank 账号时，数据中心仅会同步至修改之日起的数据到 Movebank。
